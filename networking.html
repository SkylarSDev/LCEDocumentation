Networking System

The networking system uses a packet-based client-server model. The Packet base class employs a factory pattern with idToCreateMap for packet creation. The PacketListener interface handles received packets, while Connection manages network streams. All packets implement read()/write()/handle() virtual methods. Statistics profiling is available per packet type.

Packet Base Class

Packet Abstract Base
Minecraft.World/Packet.h

Abstract base class for all network packets. Uses a factory pattern with a static ID-to-constructor map for creating packet instances by their numeric identifier. Tracks per-packet statistics and supports packet invalidation for deduplication.

PacketListener

PacketListener Interface

Interface for handling received packets. Both client and server implement this interface to process incoming packets through their respective handler methods. Each packet type's handle() method calls the corresponding method on the PacketListener implementation.

Connection

Connection Base Class

Manages network stream I/O. Base class for client and server connections. Handles reading packets from the input stream, writing packets to the output stream, and managing the lifecycle of the network connection including keepalive and disconnect handling.
