Gameplay Systems

This page covers the game mechanics systems: recipes, enchantments, containers/menus, commands, achievements, and stats. These systems work together to provide the core Minecraft gameplay experience.

Recipe System

Enchantment System

Enchantment Base Class

Enchantment
Minecraft.World/Enchantment.h

Base class for all enchantment types. Defines the enchantment's identity, weight for random selection, category for applicability, and cost range calculations per level.

Members

id (int)
Unique numeric identifier for this enchantment.
name (string)
Internal name of the enchantment.
weight (int)
Rarity weight used when randomly selecting enchantments.
category (EnchantmentCategory)
Category determining which item types this enchantment can apply to.

Methods

getMinCost(int level) → int
Returns the minimum enchanting cost required for the given enchantment level.
getMaxCost(int level) → int
Returns the maximum enchanting cost for the given enchantment level.
getMaxLevel() → int
Returns the maximum level this enchantment can reach.
canEnchant(ItemInstance) → bool
Checks whether this enchantment can be applied to the given item.
isCompatibleWith(Enchantment) → bool
Checks whether this enchantment is compatible with another enchantment on the same item.

Support Classes

EnchantmentHelper

Utility methods for applying and querying enchantments on items.

EnchantmentInstance

A specific enchantment at a specific level on an item.

EnchantmentSlot

Slot in enchanting table UI.

EnchantmentCategory

Categories: weapon, armor, tool, bow.
