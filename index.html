LCEMP Features

  • Fully working LAN multiplayer (up to 8 players)
  • Block breaking and placing synced across clients
  • Player kick system
  • Keyboard and mouse support
  • Gamma correction fixed
  • Fullscreen support

Credits

LCEMP is a Minecraft Legacy Console Edition source fork by notpies that enables LAN multiplayer alongside more complete PC compatibility. The original codebase was developed by 4J Studios for console platforms.

If you use the multiplayer code or other fixes from this project in other LCE-based projects, please credit the author and link the repository.
