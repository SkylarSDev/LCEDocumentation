UI System

Overview

LCE uses a Flash/SWF-based UI powered by Iggy/Scaleform middleware. The UILayer class manages scene navigation stacks and persistent components. Each screen is represented by a UIScene, which wraps a Flash movie object. Interactive elements are built from a UIControl hierarchy that provides buttons, sliders, lists, text inputs, and other widgets. Platform-specific IUIScene interfaces allow per-platform customization of menu behavior. The system supports multiple resolutions: 480p, 720p, 1080p, and Vita.

UIScene Base Class

UIScene
Minecraft.Client/Common/UI/UIScene.h

Base class for all UI screens. Wraps a Flash/SWF movie object via the Iggy middleware, providing lifecycle management, input handling, focus tracking, and rendering. Each scene has a loaded resolution and can draw custom slot controls for inventory item display.

UILayer

UILayer
Minecraft.Client/Common/UI/UILayer.h

Top-level UI manager that maintains a navigation stack of scenes and a set of persistent components. Handles scene transitions, focus delegation, and rendering of the entire UI layer. Supports navigating forward to new scenes and back through the stack.

UIControl Base

UIControl
Minecraft.Client/Common/UI/UIControl.h

Base class for all interactive UI elements. Each control is bound to a Flash display object via an IggyValuePath and tracks position, size, visibility, opacity, and focus state. Controls are owned by a parent UIScene and receive input through the scene's focus system.
